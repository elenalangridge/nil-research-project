# NIL Research Project

Analysis of Name, Image, and Likeness (NIL) deals at California public Division I universities (2021‚Äì2024).

**Research question:** Do social media deals command higher average NIL compensation, conditional on school, sport, and year?

**Group:** Elena Langridge, Teah Papageorgiou, Lily Proposch
**Unit:** ECC3479 ‚Äî Data and Evidence in Economics, Monash University

## How to Run the Project from Scratch

### Prerequisites ‚Äî Software

- **Python 3.8 or higher**
- **Jupyter Notebook or JupyterLab** (to run `.ipynb` files)
- **Required packages:**

| Package | Version | Purpose |
|---------|---------|---------|
| `pandas` | ‚â• 1.3 | Data loading and manipulation |
| `numpy` | ‚â• 1.21 | Numerical computation, OLS implementation |
| `scipy` | ‚â• 1.7 | Statistical distributions (t-tests, Shapiro-Wilk) |
| `matplotlib` | ‚â• 3.4 | Figures and charts |
| `jupyter` | ‚â• 1.0 | Running `.ipynb` notebooks |

Install all dependencies at once:
```bash
pip install pandas numpy scipy matplotlib jupyter
```

Or install the pinned project environment:
```bash
pip install -r requirements.txt
```

Or with a virtual environment (recommended):
```bash
python -m venv .venv
source .venv/bin/activate          # macOS/Linux
# .venv\Scripts\activate           # Windows
pip install -r requirements.txt
```

### Step-by-step Instructions

**1. Clone the repository**
```bash
git clone https://github.com/elenalangridge/nil-research-project.git
cd nil-research-project
```

**2. Create the output directories** (if running for the first time on a fresh clone)
```bash
mkdir -p data/clean results
```

**3. Run the cleaning script** ‚Äî transforms raw CSVs into a standardised transaction-level dataset
```bash
python code/01_cleaning_script.py
```
Output: `data/clean/cleaned_nil.csv` (~2,347 rows)

**4. Run the aggregation script** ‚Äî collapses transactions to school-sport-year-deal-type group averages
```bash
python code/02_aggregation_script.py
```
Output: `data/clean/nil_merged_analysis.csv` (371 rows)

**5. Run the EDA notebook** ‚Äî exploratory data analysis
```bash
jupyter notebook code/03_EDA_notebook.ipynb
```
Or execute non-interactively:
```bash
jupyter nbconvert --to notebook --execute code/03_EDA_notebook.ipynb \
  --output code/03_EDA_notebook_executed.ipynb
```

**6. Run the primary analysis notebook** ‚Äî econometric analysis, regression tables, figures
```bash
jupyter notebook code/04_primary_analysis.ipynb
```
Or execute non-interactively:
```bash
jupyter nbconvert --to notebook --execute code/04_primary_analysis.ipynb \
  --output code/04_primary_analysis_executed.ipynb
```
Outputs written to `results/`: `regression_table.png`, `coefficient_plot.png`, `residual_diagnostics.png`

**7. Run the robustness analysis notebook** ‚Äî dedicated robustness checks and comparative table
```bash
jupyter notebook code/05_robustness_checks.ipynb
```
Or execute non-interactively:
```bash
jupyter nbconvert --to notebook --execute code/05_robustness_checks.ipynb \
  --output code/05_robustness_checks_executed.ipynb
```
A pre-rendered HTML version is also available at `code/05_robustness_checks.html`.
This notebook displays a side-by-side robustness table and coefficient plot for the main social media specification.

**Run the full pipeline in one command**
```bash
python3 code/01_cleaning_script.py && \
python3 code/02_aggregation_script.py && \
python3 -m nbconvert --to notebook --execute code/04_primary_analysis.ipynb --output 04_primary_analysis_executed.ipynb --output-dir code && \
```

## Data Sources

Raw CSV files in `data/raw/` were obtained from:
- The [CalMatters NIL disclosure repository](https://github.com/calmatters/nil-disclosures)
- Individual California public university FOIA/public disclosure responses

**Raw data is included** in `data/raw/` ‚Äî no additional downloads are required.

**Schools included (12):** UC Berkeley, UCLA, UC Davis, UC Santa Barbara, UC San Diego (ucsandiego1), UC Riverside, CSU Long Beach, Cal Poly SLO, Sacramento State (4 cohorts)

**Schools excluded:**
- `fresnostate1`, `sandiegostate1` ‚Äî missing required `amount` column (excluded in cleaning script)
- Cal State Northridge, San Jose State ‚Äî reported no NIL deals (not in raw data)

## Manual Steps Outside the Code

The following steps require manual action and are **not** automated by the scripts:

1. **Data acquisition:** Raw files are committed to `data/raw/`. If you need to update with new university filings, download new CSVs to `data/raw/` and re-run the pipeline from Step 1.

2. **UC San Diego special coding rule:** Rows from `ucsandiego1` with blank `Notes` fields are coded as Social Media (`is_social_media = 1`) by assumption in `01_cleaning_script.py`. This is documented in `data/codebook.md` and is a deliberate analytical choice.

3. **Data validation:** After running the scripts, verify row counts in `data/clean/`:
   - `cleaned_nil.csv` should have ~2,347 rows
   - `nil_merged_analysis.csv` should have 371 rows (the notebook drops 13 partial-year 2025 rows internally, leaving N=358 for analysis)

4. **Results directory:** The `results/` folder must exist before running the primary analysis notebook. If missing: `mkdir -p results`

## Key Analytical Choices

| Choice | Rationale |
|--------|-----------|
| Log(1 + value) as outcome | Raw values severely right-skewed (skewness ‚âà 8.5); log yields near-normal residuals |
| Drop 2025 observations (N=13) | Partial-year reporting; done inside the notebook, not the cleaning script |
| School-clustered SEs in M3‚ÄìM4 | Correct for within-school correlation across sport-year cells |
| Descriptive (not causal) declaration | No credible instrument or natural experiment available |
| `is_social_media` by keyword matching | Automated classification from free-text `deal_description` field |

See `data/codebook.md` for full variable definitions, and `code/04_primary_analysis.ipynb` Section 7 for a detailed discussion of threats to validity.
