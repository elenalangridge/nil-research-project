# NIL Social Media Impact Analysis

### Repository Structure
- **data/raw/**: Contains the original university CSV files (e.g., `ucla.csv`, `ucdavis.csv`) [3-8].
- **data/clean/**: Contains the final merged dataset (`nil_merged_analysis.csv`) and the data codebook.
- **code/**: Python scripts for data cleaning and aggregation.

### Software Requirements
- Python 3.x
- `pandas`
- `numpy`
