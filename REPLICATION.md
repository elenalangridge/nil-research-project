## Data Pipeline Explained

### Step 1: Data Cleaning (`code/01_cleaning_script.py`)

**Input:** Raw CSV files in `data/raw/` (one per university)

**Output:** `data/clean/cleaned_nil.csv` (~2,347 transaction-level rows)

**What it does:**
- Standardises column names across universities
- Parses deal amounts and dates
- Classifies deals as social media or traditional using keyword matching
- Handles missing/invalid data
- Applies deterministic random seed for reproducibility

**Key analytical decision:** Rows from `ucsandiego1` with blank `Notes` fields are coded as social media deals.

### Step 2: Aggregation (`code/02_aggregation_script.py`)

**Input:** `data/clean/cleaned_nil.csv`

**Output:** `data/clean/nil_merged_analysis.csv` (371 rows)

**Aggregation structure:** school × sport × year × deal_type

**Observations created:** Average transaction value per (school, sport, year, social media status) cell.

**Filtering:** Drops 13 partial-year 2025 observations in downstream analysis (done in notebooks, not here).

### Step 3: Primary Analysis (`code/04_primary_analysis.ipynb`)

**Input:** `data/clean/nil_merged_analysis.csv`

**Main specification (M4):**
```
log(1 + value) = β₀ + β₁·social_media + year_fe + school_fe + sport_fe + ε
```

**Estimation:**
- OLS with HC3 robust standard errors
- Standard errors clustered at the school level (G = 12 clusters)
- Filters to year ≤ 2024 (N = 358 observations)

**Key output:** Coefficient table and Figure 1 (saved to `results/regression_table.png`)

### Step 4: Robustness Checks (`code/05_robustness_checks.ipynb`)

**Input:** `data/clean/nil_merged_analysis.csv`

**Specifications tested:**
1. **M3:** Drop sport FE (year + school only)
2. **Drop UCSD:** Rerun M4 excluding `ucsandiego1`
3. **Winsorize 95%:** Rerun M4 on log outcome winsorized at 95th percentile
4. **Levels:** Rerun M4 on raw (untransformed) transaction values
5. **Two-way clustering:** Rerun M4 with school+sport clustered SEs

**Output:** Robustness table (printed in notebook) and Figure 2 (saved to `results/coefficient_plot.png`)
